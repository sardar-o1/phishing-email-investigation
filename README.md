# phishing-email-investigation

## Lab Objective

This project demonstrates a structured approach to investigating phishing emails using real-world email samples collected from public research repositories. Each investigation follows a SOC analyst workflow, including email header analysis, sender verification, authentication review, IOC extraction, threat intelligence enrichment, and incident reporting.

The primary objective of this lab is to develop practical phishing investigation skills that are essential for an entry-level Security Operations Center (SOC) Analyst role.
